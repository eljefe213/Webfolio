Project Overview

The project involved creating a website for a Market Coffee using the programming languages HTML, CSS, and JavaScript. We started by creating a mockup using Adobe XD to define the design and layout of the site.

We then used HTML to structure the site's content using tags for headers, paragraphs, images, links, and buttons.

We then used CSS to style the site, adding colors, fonts, borders, and backgrounds to give the site a professional and appealing look. We also used CSS layout techniques to align the site's elements and create visual effects such as transitions and animations.

Finally, we used JavaScript to add interactive features such as dropdown menus, contact forms, image slideshows, and social media share buttons. We also used JavaScript libraries such as jQuery to simplify the development process and add additional features to the site.

By combining HTML, CSS, and JavaScript, we created a website for a Market Coffee that is both functional, aesthetic, and interactive, thus providing a pleasant and immersive user experience.
